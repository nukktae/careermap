# Toss Design Principles Research

## Key Insights from Toss's 10-Year Design Evolution

### Core Design Philosophy
**"Design's core purpose is solving fundamental user problems, not merely aesthetics."**

Toss established early design principle: **Simplicity and intuitive experiences**

### Evolution Stages

#### 1. Early Days: Problem-Solving Over Polish
- Focused on core innovation (simplified bank transfers)
- Prioritized radical UX improvements over visual perfection
- Powerful user experience can overcome visual imperfections

#### 2. Growth Era: Systematic Design Capability

**Project 3.0 - Comprehensive Evolution:**
- Design systems
- Branding
- UX writing
- Qualitative research
- Graphic technologies

**UX Writing Impact:**
- Boosted conversion by 300%
- Text is foundational element of UX, not just content
- Data-driven UX writing enhances product performance

**In-house Graphic Excellence:**
- Designers mastered Cinema4D
- Evolved from simple Lottie animations to sophisticated real-time interactions
- Expanding designers' skillsets boosts productivity and quality

**Qualitative User Testing (UT):**
- Built internal UT culture
- Uncovered insights beyond A/B tests
- Significantly improved revenue

#### 3. Advanced UX: Aesthetics + Platform Thinking

**MyData Era Adaptation:**
- Home screen evolved into platform guiding financial behavior
- Beyond mere information display

**Funnel Optimization:**
- Systematically optimized common funnels across financial domains
- Elevated collective conversion rates
- Systematize repeating UX patterns into shared design knowledge

**Brand Evolution:**
- Shifted from casual to trustworthy tech branding
- Experimented with 3D tools for visual storytelling

#### 4. Current: AI Integration + Interaction Design

**AI-Powered Design:**
- Huribot: AI-driven heuristic evaluations
- Tosst: AI-generated Toss-style graphics
- Automate repetitive tasks, enhance creativity

**Interaction Innovation:**
- Designers directly shape interactions
- Developers rapidly implement via tokenized languages
- Interaction design essential for conversions

### Key Takeaways for Career Platform

1. **Solve problems first, polish later**: Focus on core user pain points
2. **UX writing is critical**: Clear, data-driven copy can boost conversions dramatically
3. **Systematic approach**: Build design systems, not just individual screens
4. **Qualitative research**: User testing uncovers insights that data alone misses
5. **Continuous evolution**: Design is ongoing learning, not a destination
6. **Interaction matters**: Micro-interactions and animations drive engagement
7. **AI as tool**: Use AI to automate repetitive tasks and enhance creativity
8. **Platform thinking**: Design holistic experiences, not isolated features
